# Projet_-Python_pour_la_Data_Science
Ce répertoire regroupe l’ensemble des livrables réalisés dans le cadre du projet du cours de Python pour la data science. Il inclut les différentes étapes du travail, notamment le nettoyage des données, l’analyse exploratoire ainsi que les résultats et interprétations associés.
Les modules permettent l'organisation du code. 
Dans les modules on retrouve les fonctions de calcul de la surreprésentation des candidats et de la cartographie propre à un candidat.
